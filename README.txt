前言：
B站up主：三上烤鸭 自制大气层整合包

安装说明：
确保你的 Nintendo Switch 已经破解并运行最新版 CFW。
从本项目的 Releases 页面下载最新版本。
将下载的文件解压缩到 SD 卡的根目录中。
按照常规流程启动您的设备进入 CFW 模式。

使用说明：使用二进制脚本Rcleaner进行一键清理后，解压此文件夹并将该文件夹放在您的内存卡根目录处！

提醒：
这里简单提醒一下：所有获得烤鸭包最新版本使用权力的人，都可以自由转载本包给任何你需要的人。
